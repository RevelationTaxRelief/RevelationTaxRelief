## ğŸ› Troubleshooting

### Page Not Found (404)
- Verify file is in correct folder: `app/route-name/page.tsx`
- Check for typos in route name
- Ensure export is `export default function ComponentName()`

### Styling Not Applied
- Check file starts with `"use client"`
- Verify Tailwind classes are spelled correctly
- Check globals.css for color definitions
- Clear build cache: `rm -rf .next`

### Build Fails
- Run `pnpm install` to ensure dependencies
- Check for TypeScript errors: `pnpm type-check`
- Verify no import errors
- Check Node.js version compatibility
